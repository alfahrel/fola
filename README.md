# Fola

A simple task manager for Android with an Indonesian holiday calendar.

## Features

- Add, edit, and delete tasks
- Filter tasks by category (Task / Habit) and date
- Home screen widget for your task list
- Built-in Indonesian holiday calendar with swipeable months
- Calendar home screen widget with prev/next month navigation
- Dynamic color (Material You)

## Requirements

- Android 8.0+

## Notes

This is an beta release. Expect bugs and missing features.
